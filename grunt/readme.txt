项目开发任务，自动压缩src文件夹中的html、css、js至dist文件夹，修改保存后自动刷新浏览器

1.需要node.js环境

2.安装grunt-cli
>npm install -g grunt-cli

3.进入项目根目录，复制Gruntfile.js和package.json至根目录，cmd安装grunt
cd 项目根目录
>npm install

4.浏览器安装livereload插件(http://feedback.livereload.com/knowledgebase/articles/86242-how-do-i-install-and-use-the-browser-extensions-)，cmd启动监控
>grunt live
